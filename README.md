
# FlyingThanos

Game based on [Flying Hero](https://youtu.be/zr4A0Rsxg1M) by Björn Granberg Ahlmark.



![game](/github/game.gif)



## Features

- 7 different modes

- 6 different skins

- 4 maps with unique music

- Detailed [stats & highscores](/github/stats.png)

- osu! beatmap support

- Local multiplayer



### Modes

- **Classic** - normal flying hero gameplay

- **Swarm** - more and faster shurikens

- **Refuel** - collect items to refuel (survive)

- **Sharp Shurikens** - unlimited time, but 3 hits and you're out

- **Flying Fast** - faster items and more points, but less time

- **Duel** - classic but with two players

- **Catch** - mode inspired by [osu!catch](https://osu.ppy.sh/home) with support for most beatmaps
